# Credit Scoring Pipeline (German Credit)

Production-ready ETL-пайплайн и модель предсказания невозврата кредита на базе набора данных German Credit (UCI). Проект реализует полный цикл от сырых данных до обученной и настроенной модели.

## Архитектура

- **SQL (PostgreSQL)** — Хранение сырых данных, Feature Engineering (бининг, WOE/Bad Rate), формирование витрины для модели.
- **Python** — Чтение витрины, обучение RandomForest, подбор порога классификации под бизнес-требования, оценка и сохранение модели.
- **Git** — контроль версий кода и модели.

## Структура репозитория

| Файл | Назначение |
|---|---|
| `create_model_dataset.sql` | SQL-запросы и функции для создания витрины данных `model_dataset` |
| `german_credit.ipynb` | Jupyter-ноутбук: загрузка данных в Pandas, обучение модели, настройка порога, сохранение артефакта |
| `credit_scoring_model.pkl` | Обученная модель (RandomForest) и выбранный порог вероятности |
| `.gitignore` | Исключения для служебных файлов |
